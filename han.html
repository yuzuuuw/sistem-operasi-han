04 / Virtual Memory

Virtual Memory & Paging

Simulasikan bagaimana OS memperluas RAM menggunakan disk

🔮

Virtual Address Space

Program "melihat" memori yang jauh lebih besar dari RAM fisik. OS mengelola pemetaan page virtual ke frame fisik.

📄

Paging

Memori dibagi menjadi blok kecil (pages). Tidak semua pages harus ada di RAM sekaligus — sisanya di disk.

⚡

Page Fault

Ketika CPU mengakses page yang tidak ada di RAM, terjadi "page fault". OS harus swap page dari disk ke RAM.
